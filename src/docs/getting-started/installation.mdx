## Simple dApp

First, you need to install the `@airgap/beacon-sdk` package.

<Tabs
groupId="packet-manager"
defaultValue="npm"
values={[
{ label: "NPM", value: "npm" },
{ label: "Yarn", value: "yarn" },
]}
>

<TabItem value="npm">

```bash
npm install --save @airgap/beacon-sdk
```

</TabItem>
<TabItem value="yarn">

```bash
yarn add @airgap/beacon-sdk
```

</TabItem>
</Tabs>

## dApp with Contract Interaction

If you plan to interact with contracts, you will most likely instead want to
install `@taquito`. Taquito installs the `beacon-sdk` as a dependency, so it
doesn't have to be installed manually.

<Tabs
  groupId="packet-manager"
  defaultValue="npm"
  values={[
    { label: "NPM", value: "npm" },
    { label: "Yarn", value: "yarn" },
  ]}
>
<TabItem value="npm">

```bash
npm install --save @taquito/taquito @taquito/beacon-wallet
```

</TabItem>
<TabItem value="yarn">

```bash
yarn add @taquito/taquito @taquito/beacon-wallet
```

</TabItem>
</Tabs>
